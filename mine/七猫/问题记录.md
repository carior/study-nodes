## 作家专区本地登陆不上

```
/**
 * 本地 https://localhost 开发时，上游 Set-Cookie 若带 Domain=.qimao.com 等，
 * 浏览器会拒绝写入；通过代理改写为 localhost，author-token 等 Cookie 才能生效。
 * 参见 http-proxy / webpack-dev-server 的 cookieDomainRewrite。
 */
const DEV_PROXY_COOKIE_DOMAIN_REWRITE = { cookieDomainRewrite: 'localhost' };

let devProxyConfig = {}
/*** 本地联调用，将路由代理至开发本地 ***/
if (process.env.VUE_APP_ENV === 'location') {
    const localApiOrigin = {
        devOrigin: 'http://172.16.76.48:8094/',
        apis: [
            // '/pc/v1/profit/book-month-profit'
        ]
    }
    for (const api of localApiOrigin.apis) {
        devProxyConfig[`/apiQM${api}`] = {
            target: localApiOrigin.devOrigin,
            changeOrigin: true,
            secure: false,
            pathRewrite: {
                '^/apiQM': 'api'
            },
            ...DEV_PROXY_COOKIE_DOMAIN_REWRITE,
        }
        devProxyConfig[`/apiZongheng${api}`] = {
            target: localApiOrigin.devOrigin,
            changeOrigin: true,
            secure: false,
            pathRewrite: {
                '^/apiZongheng': 'api'
            },
            ...DEV_PROXY_COOKIE_DOMAIN_REWRITE,
        }
    }
    console.log('devProxyConfig ==> ', devProxyConfig)
}


'/apiQM': {
      target: 'https://zuozhe.qimao.com',
      changeOrigin: true,
      secure: false,
      pathRewrite: {
          '^/apiQM': 'api'
      },
      ...DEV_PROXY_COOKIE_DOMAIN_REWRITE
  },
```

原因：本地代理到服务端的服务器上了

解决方式（2选1）：

1. 注释这段代码
2. 设置cookieDomainRewrite: 'localhost'
