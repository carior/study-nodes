## 验证规则

### 必填字段验证

```JavaScript
rules: {
    sign_type: DEFAULT_RULE,                    // 合同形式
    contract_type: DEFAULT_RULE,                // 合同类型
    price: INTEGERPLUSZERO_RULE,               // 价格
    first_scripts_time: DEFAULT_RULE,           // 交稿时间
    except_month_words: INTEGERPLUSZERO_RULE,   // 每月字数
    book_length_type: DEFAULT_RULE,             // 篇幅类型
    except_total_words_max: INTEGERPLUSZERO_RULE, // 最大字数
    except_total_words_min: INTEGERPLUSZERO_RULE, // 最小字数
    guarding_right_proportion: DEC_RULE,        // 维权分成
    our_calculate_proportion: DEC_RULE,         // 自有平台分成
    calculate_proportion: DEC_RULE,             // 第三方分成
    nonele_calculate_proportion: DEC_RULE,      // 版权分成
    book_level: DEFAULT_RULE,                   // 作品分级
    copyright_list: DEFAULT_RULE,               // 版权类型
    contract_term_type: INTEGERPLUSZERO_RULE,   // 授权期限
    contract_no: DEFAULT_RULE,                  // 合同编号
    author_sign_time: DEFAULT_RULE,             // 签署日期
    calculate_proportion_term: DEFAULT_RULE,     // 结算期限
}
```

### 验证规则类型

- `DEFAULT_RULE` - 基础必填验证
- `INTEGERPLUSZERO_RULE` - 正整数验证（包含0）
- `DEC_RULE` - 小数验证
