## 一、安装 Go SDK

### macOS

```bash
# 推荐用 Homebrew 安装
brew install go

# 验证安装
go version
# 输出类似：go version go1.22.x darwin/arm64
```

### Windows

1. 下载安装包：[go.dev/dl](https://go.dev/dl/)
2. 双击安装，一路下一步
3. 打开 PowerShell，执行 `go version` 验证

### Linux

```bash
# 下载二进制包
wget https://go.dev/dl/go1.22.linux-amd64.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.linux-amd64.tar.gz

# 把 /usr/local/go/bin 加入 PATH（写入 ~/.bashrc 或 ~/.zshrc）
export PATH=$PATH:/usr/local/go/bin

# 验证
go version
```

## 二、配置环境变量

Go 默认工作目录是 `$HOME/go`，可以通过 `GOPATH` 自定义：

```bash
# 查看当前配置
go env GOPATH
go env GOROOT
go env GOPROXY   # 模块代理，推荐用国内镜像

# 设置 GOPROXY（国内加速）
go env -w GOPROXY=https://goproxy.cn,direct
```

> [!tip] 常见 Go env 变量
> | 变量 | 作用 |
> |------|------|
> | `GOPATH` | 工作空间根目录（默认 `~/go`）|
> | `GOROOT` | Go 安装目录（一般不改）|
> | `GOPROXY` | 模块下载代理 |
> | `GO111MODULE` | 是否启用模块（`on`=模块模式，`auto`=自动）|

## 三、配置 VS Code

### 1. 安装插件

在 VS Code 扩展商店搜索 `Go`，安装由 **Go Team at Google** 出品的官方插件。

### 2. 安装工具链

打开命令面板（`Cmd/Ctrl + Shift + P`），输入 `Go: Install/Update Tools`，全选所有工具并安装：

| 工具            | 作用                         |
| ------------- | -------------------------- |
| `gopls`       | Language Server，代码补全/跳转/诊断 |
| `go-outline`  | 大纲视图                       |
| `gorename`    | 重命名重构                      |
| `delve`       | 调试器                        |
| `staticcheck` | 静态分析                       |

> [!warning] 第一次装插件后必须执行这一步
> 否则代码不会飘红（错误提示）、无法跳转、无法调试。

### 3. 推荐 VS Code 设置

```json
{
  "go.useLanguageServer": true,
  "go.formatTool": "gofmt",
  "go.lintTool": "staticcheck",
  "editor.formatOnSave": true
}
```

## 四、创建第一个程序

### 1. 创建项目目录

```bash
mkdir -p ~/go/src/hello
cd ~/go/src/hello
```

### 2. 初始化 Go 模块（必须）

```bash
go mod init hello
# 会在当前目录生成 go.mod 文件
```

### 3. 编写 Hello World

创建文件 `main.go`：

```go
package main

import "fmt"

func main() {
	fmt.Println("Hello, World!")
}
```

### 4. 运行

```bash
# 方式一：直接运行（编译临时文件并执行）
go run main.go

# 方式二：编译为可执行文件
go build -o hello main.go
./hello
```

> [!success] 验收标准
> 终端输出 `Hello, World!`，无任何报错。

## 五、常用命令速查

| 命令 | 作用 |
|------|------|
| `go run main.go` | 编译并运行（不生成二进制）|
| `go build -o name main.go` | 编译为可执行文件 |
| `go mod init module名` | 初始化模块 |
| `go mod tidy` | 整理依赖 |
| `go get 包名` | 下载依赖包 |
| `go test ./...` | 运行所有测试 |

## 六、常见问题

> [!faq]- go: command not found
> PATH 没配好，确认 `export PATH=$PATH:/usr/local/go/bin` 已写入 shell 配置文件，并执行 `source ~/.zshrc` 或重启终端。

> [!faq]- 插件装完不飘红 / 没有补全
> 执行 `Cmd/Ctrl + Shift + P` → `Go: Install/Update Tools` → 全选 → 回车，等待安装完成。

> [!faq]- go mod init 报错
> 确保当前目录是空目录，或者手动指定模块名：`go mod init github.com/你的用户名/项目名`

## 七、本项目 Go 工具链配置

> [!warning] 本节仅适用于本项目（drsp）的 Go 服务端
> 包含私有模块访问、Protoc、hosts 等项目特定配置。学习通用 Go 语法可跳过本节。

### 1. Go 版本管理（推荐）

使用 `g` 工具管理多个 Go 版本：

```bash
# 安装 g（macOS/Linux）
curl -sSL https://github.com/voidint/g/releases/download/v1.5.0/g1.5.0.darwin.arm64.tar.gz | tar -xz -C /usr/local

# 或参考官方文档：https://github.com/voidint/g
```

### 2. 配置私有模块访问

本项目使用阿里云私有模块仓库 `codeup.aliyun.com`，需配置环境变量：

```bash
# 配置私有模块路径
go env -w GOPRIVATE=codeup.aliyun.com
go env -w GONOPROXY=codeup.aliyun.com
go env -w GONOSUMDB=codeup.aliyun.com

# 验证
go env | grep -E 'GOPRIVATE|GONOPROXY|GONOSUMDB'
```

### 3. 配置 .netrc（凭据文件）

```bash
# 创建文件
touch ~/.netrc

# 写入凭据（用于访问 codeup.aliyun.com）
# 格式：machine {host} login {username} password {password}
machine codeup.aliyun.com login <用户名> password <密码>
```

> [!warning] 凭据安全
> `.netrc` 包含明文密码，确保文件权限正确：`chmod 600 ~/.netrc`

### 4. Protoc 编译工具

```bash
# macOS 安装（较慢，可配置代理）
brew install protobuf

# 验证安装
protoc --version
# 输出：libprotoc 23.3
```

> [!tip] 如果 brew 安装超时
> 手动下载：[github.com/protocolbuffers/protobuf/releases](https://github.com/protocolbuffers/protobuf/releases)
> 选择对应平台的 `protoc-{version}-linux-osx.zip`，解压后将 `bin/protoc` 放入 PATH。

### 5. hosts 配置

```bash
# 追加到 /etc/hosts
47.117.64.60 user-manage-center.qmniu.com
47.102.128.38 api-original-platform.qimao.com
```

> [!tip] 快速编辑 hosts
> ```bash
> sudo vim /etc/hosts
> # 或用 SwitchHosts 工具管理 hosts，更方便
> ```

### 6. 启动项目服务

```bash
# 进入服务端项目目录
cd <服务端项目路径>

# 启动 HTTP 服务
go run main.go http
```

> [!success] 启动成功后
> 服务监听 `127.0.0.1:8087`，前端代理配置见 [[本项目前端本地开发指南]]。
