## 一、变量与常量

### 变量声明

```go
// 标准写法（类型推导）
name := "张三"     // 自动推断为 string
age := 28         // 自动推断为 int

// 显式类型
var name string = "张三"
var age int = 28

// 声明多个变量
var (
    name string
    age  int
)

// 函数内简写
x, y := 1, 2
```

> [!tip] 和前端 JS 对比
> `:=` 类似 JS 的 `let` / `const`，但**只在函数内使用**，不能用于全局。
> 全局变量必须用 `var` 关键字。

### 常量

```go
const Pi = 3.14159
const (
    StatusOK    = 200
    StatusNotFound = 404
)
```

## 二、数据类型（速记）

| 类型 | 示例 | 说明 |
|------|------|------|
| `int` / `int64` | `10` | 整数 |
| `float64` | `3.14` | 浮点数（没有 float32/64 混用）|
| `string` | `"hello"` | 字符串（不可变）|
| `bool` | `true` / `false` | 布尔 |
| `[]int` | `[]int{1, 2, 3}` | Slice（动态数组）|
| `map[string]int` | `map[string]int{"a": 1}` | 哈希表 |
| `struct` | 见结构体章节 | 结构体 |

> [!warning] Go 没有 `null` / `undefined`
> Go 使用**零值**（zero value）作为默认值：
> - 数值：`0`
> - 字符串：`""`
> - 布尔：`false`
> - 指针/接口：`nil`
> 用 `nil` 而不是 `null`！

## 三、控制流

### if 条件

```go
if score := 85; score >= 60 {
    fmt.Println("及格")
} else {
    fmt.Println("不及格")
}
```

> [!tip] 和 JS 的区别
> Go 的 `if` 条件**不需要括号**，且支持条件前加初始化语句（`score := 85;`）。

### for 循环（Go 只有 for）

```go
// 标准 for 循环（类似 JS 的 for）
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// 充当 while
n := 5
for n > 0 {
    n--
}

// 无限循环
for {
    // ...
    break
}

// range 遍历（类似 JS 的 forEach）
nums := []int{1, 2, 3}
for i, v := range nums {
    fmt.Printf("index: %d, value: %d\n", i, v)
}
```

### switch

```go
day := "Monday"

switch day {
case "Monday", "Tuesday":
    fmt.Println("工作日")
case "Saturday", "Sunday":
    fmt.Println("周末")
default:
    fmt.Println("未知")
}
```

> [!tip] 和 JS 的区别
> - 不需要 `break`，默认自动 break
> - 要继续往下执行？用 `fallthrough` 关键字
> - `switch` 后可以不跟变量，等同于多个 `if-else`

## 四、函数

```go
// 标准函数
func add(a int, b int) int {
    return a + b
}

// 多返回值（Go 特色）
func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("除数不能为零")
    }
    return a / b, nil
}

// 命名返回值
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return // 直接返回命名变量
}
```

## 五、错误处理

Go 没有 `try-catch`，用**返回值**传递错误：

```go
result, err := divide(10, 0)
if err != nil {
    fmt.Println("出错了:", err)
    return
}
fmt.Println("结果:", result)
```

> [!example] 对比前端
> Go 的 `error` 类似前端的 `try-catch`，但通过**返回值**传递，更显式。
> 很多 Go 代码会看到 `if err != nil { return err }` 这种模式。

## 六、Map 与 Slice

### Map（类似 JS 的对象/Map）

```go
// 创建
ages := make(map[string]int)
ages["张三"] = 25
ages["李四"] = 30

// 读取
age := ages["张三"]

// 判断 key 是否存在
if v, ok := ages["王五"]; ok {
    fmt.Println(v)
} else {
    fmt.Println("key 不存在")
}

// 删除
delete(ages, "张三")
```

### Slice（类似 JS 的数组）

```go
// 创建
nums := []int{1, 2, 3, 4, 5}

// 添加元素
nums = append(nums, 6)

// 切片（类似 JS 的 slice）
sub := nums[1:4]  // [2, 3, 4]

// 长度和容量
len := len(nums)
cap := cap(nums)
```

## 九、指针（速记）

Go 支持指针，但没有指针运算：

```go
n := 10
p := &n        // 取地址：p 是 *int 类型
fmt.Println(*p)  // 解引用：输出 10

// 函数修改外部变量（Go 的"引用传递"）
func inc(x *int) {
    *x++
}
inc(&n)
```

> [!tip] 什么时候用指针？
> - 函数内想修改外部变量 → 用指针
> - 传递大结构体避免拷贝 → 用指针
> - 其他情况用值类型即可
