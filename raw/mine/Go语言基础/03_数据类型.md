# Go 数据类型详解

> [!info] 章节说明
> 本章深入 Go 的数据类型体系，重点理解 **值类型 vs 引用类型**，这是阅读源码时理解行为的关键。

---

## 一、Go 的类型分类

```
Go 类型
├── 值类型（Value Types）
│   ├── int / float / bool / string
│   └── struct
│
└── 引用类型（Reference Types）
    ├── slice
    ├── map
    ├── channel
    ├── interface
    └── pointer（指针本身是值，但指向的是对象）
```

> [!tip] 核心区别
> - **值类型**：变量直接存值，赋值时**拷贝整个值**
> - **引用类型**：变量存的是地址（类似指针），赋值时**拷贝地址**

---

## 二、值类型详解

### 基本类型

```go
var a int = 10
var b int = a  // b 是 a 的副本，修改 b 不影响 a
b = 20
fmt.Println(a) // 10，不变
fmt.Println(b) // 20
```

### struct（值类型）

```go
type User struct {
    Name string
    Age  int
}

u1 := User{Name: "张三", Age: 25}
u2 := u1          // 拷贝整个结构体！
u2.Age = 30
fmt.Println(u1.Age) // 25，不变
fmt.Println(u2.Age) // 30
```

## 三、引用类型详解

### Slice（动态数组）

> [!warning] 重点理解：Slice 由三部分组成
> Slice 内部是一个数组的引用，由三部分构成：
> ```
> Slice = { ptr(指向数组), len(长度), cap(容量) }
> ```
> 对 Slice 做 `append` 可能触发扩容（新建底层数组），老 Slice 不受影响。

```go
// 创建 Slice
nums := []int{1, 2, 3}
nums2 := nums       // 共享同一个底层数组！
nums2[0] = 99
fmt.Println(nums)  // [99, 2, 3] ← nums 也被改了！

// 用 make 创建（明确容量）
s := make([]int, 0, 10)

// append
s = append(s, 1, 2, 3)

// 切片（共享底层数组）
sub := s[0:2]
```

### Map

```go
// 创建 Map
m := make(map[string]int)

// 赋值
m["a"] = 1
m["b"] = 2

// 读取
v := m["a"]

// 判断 key 是否存在
if v, ok := m["c"]; ok {
    fmt.Println("存在:", v)
} else {
    fmt.Println("key 不存在")
}

// 删除
delete(m, "a")

// 遍历
for k, v := range m {
    fmt.Printf("%s -> %d\n", k, v)
}
```

> [!tip] Map 的零值
> Map 的零值是 `nil`，不能直接使用（读写会 panic）。
> 必须用 `make(map[string]int)` 初始化后才能用。

### Pointer（指针）

```go
n := 10
p := &n          // p 类型是 *int
fmt.Println(*p)  // 解引用：10

*p = 20          // 通过指针修改 n
fmt.Println(n)  // 20

// new 函数：创建指针并初始化
p2 := new(int)   // *int，值为 0
*p2 = 100
```

## 四、类型转换（无隐式转换）

Go 不会自动转换类型，必须**显式调用转换函数**：

```go
var i int = 42
var f float64 = float64(i)  // int → float64
var u uint = uint(f)         // float64 → uint

// 字符串和字节切片
str := "hello"
bytes := []byte(str)
str2 := string(bytes)

// 字符串和数字（需要 strconv 包）
num := 123
str3 := strconv.Itoa(num)        // int → string
num2, _ := strconv.Atoi(str3)   // string → int
```

> [!warning] 和前端 JS 的巨大差异
> JS 会自动类型转换（`'1' + 1 → '11'`），Go **绝不自动转**，必须显式写。
> 这让 Go 更安全，但写代码时要多注意。

## 五、字符串（不可变）

```go
s := "hello 世界"

// 长度（字节数，不是字符数）
len(s)  // 12（UTF-8 中文字符占 3 字节）

// 遍历字符
for i, r := range s {
    fmt.Printf("%c", r)  // h e l l o  世界
}

// 子串
sub := s[0:5]  // "hello"

// 拼接
s2 := s + "!"  // "hello 世界!"
```

## 六、自定义类型

```go
// type 别名（新类型，不兼容底层类型）
type ID int
type Handler func(int) bool

// type 定义（基于已有类型创建新类型）
type User struct {
    Name string
    Age  int
}

// 类型方法绑定（见结构体章节）
func (u User) Greet() string {
    return "你好，我是" + u.Name
}
```

## 七、类型断言（接口相关）

当变量是接口类型时，可以用类型断言提取具体类型：

```go
var i interface{} = "hello"

// 安全断言（返回 value, ok）
if s, ok := i.(string); ok {
    fmt.Println("是字符串:", s)
} else {
    fmt.Println("不是字符串")
}

// 不安全断言（panic）
s2 := i.(string) // 如果是其他类型，会 panic
```

## 八、与前端 JS 对比速查

| 概念 | Go | 前端 JS |
|------|-----|---------|
| 整数 | `int` / `int64` | `Number`（整数小数统一）|
| 字符串 | `string`（不可变）| `string`（可变）|
| 数组 | `[]int`（固定类型）| `Array<any>`（任意类型）|
| 对象 | `struct` / `map` | `Object` / `Map` |
| 引用传递 | Slice / Map / Pointer | Object / Array |
| 类型转换 | 必须显式 `float64(i)` | 自动隐式转换 |
| `null` | `nil` | `null` / `undefined` |
