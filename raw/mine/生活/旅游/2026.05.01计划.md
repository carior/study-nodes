# 五一上海 3 日悠闲游计划（不住宿｜适合老人 & 2 岁以下宝宝）

> 核心原则：**上午玩、中午吃、下午早回**；整体不赶路，优先照顾老人和宝宝的体力节奏。

## 出行备忘清单

### 宝宝用品

- [ ] 挖沙工具套装
- [ ] 换洗衣物（至少 1 套）
- [ ] 便携式辅食 / 零食
- [ ] 保温水壶
- [ ] 遮阳帽
- [ ] 湿巾 / 尿不湿 / 隔尿垫
- [ ] 婴儿车（一定要带，青西和庄行都用得上）

### 老人用品

- [ ] 折叠小马扎（排队或休息时用）
- [ ] 常用药品（降压药、晕车药等）
- [ ] 薄外套（郊野比市区凉）
- [ ] 老花镜 / 手机（方便看导航和拍照）

### 通用 / 车上常备

- [ ] **大瓶驱蚊液**（青西最需要）
- [ ] 充电宝 + 充电线
- [ ] 零食 + 饮用水
- [ ] 纸巾 / 湿巾（大包装）
- [ ] 垃圾袋（车上和野餐用）
- [ ] 防晒霜、遮阳伞 / 帽子

## 每日节奏参考

| 时间点 | Day 1（市区公园） | Day 2（奉贤） | Day 3（青西） |
| :-- | :-- | :-- | :-- |
| 8:30 | 出门 | 出门 | **必须出门**（9 点前到） |
| 9:30 | 到达公园开始玩 | 到达采摘园 | 到达公园，坐观光车 |
| 11:30 | 准备去附近商场 | 采摘结束 | 水上森林徒步 |
| 12:00–13:30 | 商场午餐 + 休息 | 一品羊庄午餐 | 简单野餐或农家乐 |
| 13:30–15:00 | 回家午睡 | 庄行花屿泛舟 / 草坪休息 | 继续逛或坐车返回 |
| 15:00–16:00 | 在家休息 | 返程（约 1 小时） | 返程（错开晚高峰） |

> 这套安排按“上午玩、中午吃、下午早回”的节奏设计，老人和孩子都比较容易接受。  
> 如果需要调整某一天的项目，或者想换备选地点，可以再继续改。
