### 四、家访当天额外提醒

**关于监控的提问技巧**：
- 不要一上来就问“监控能不能随时看”，容易让老师觉得你不信任。
- 建议这样说：**“如果孩子在园里发生特殊情况，我们怎么了解经过？监控方面有什么流程可以查看吗？”**
- 这样既表达了关心，又显得尊重园方的规则。

**除了“问”，更要“看”和“听”**：

| 看什么 | 听什么 |
| :--- | :--- |
| 老师蹲下跟孩子说话时语气是否耐心 | 老师回答含糊其辞时，是否在回避 |
| 角落里的教具是否有灰尘（判断日常维护） | 提到“分离焦虑”时，老师给出的方案是具体的还是泛泛的 |
| 卫生间是否干净、有无异味 | 问“冲突处理”时，老师说的是“惩罚”还是“引导” |
| 整体氛围是温暖有序还是压抑混乱 | 提到“过敏备餐”时，回答是肯定的还是“应该可以” |

**相信你的直觉**：如果某个回答让你不舒服，大概率以后会更不舒服。
