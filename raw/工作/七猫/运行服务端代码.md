### 前端的修改

vite.config.ts

```
server: {
      ...(_common as any).server,
      proxy: {
          // /api/api/* 全部转发到本地 Go 服务，rewrite 去掉多余的 /api 前缀
          '/api/api/copyright': {
               target: 'http://127.0.0.1:8087',
               changeOrigin: true,
               rewrite: (path) => path.replace(/^\/api\/api\/copyright/, '/api/copyright'),
          },
          // /api/site、/api/manage 等转发到线上
          '/api': {
              target: 'https://drsp.qmniu.com',
              changeOrigin: true,
              rewrite: (path) => path.replace(/^\/api/, ''),  // 去掉 /api 前缀
          },
      },
  },
```

